# Privacy Policy for Roboto SAI 2026

**Last Updated:** February 11, 2026

## Overview
Roboto SAI 2026 ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our AI-powered platform.

## Information We Collect

### Personal Information
- **Account Data:** Email address, name (if provided during registration)
- **Usage Data:** Chat messages, session data, API interactions
- **Technical Data:** IP address, browser type, device information

### AI-Generated Data
- Conversation summaries and rollups
- Semantic embeddings for search functionality
- Emotion analysis metadata

## How We Use Your Information

### Core Service Delivery
- Process and respond to chat requests
- Generate conversation summaries
- Provide AI-powered code assistance

### Platform Improvement
- Train and improve AI models (anonymized data only)
- Monitor system performance and security
- Debug and troubleshoot issues

### Legal Compliance
- Maintain audit logs for security
- Comply with applicable regulations
- Protect against fraud and abuse

## Data Storage and Security

### Encryption
- All data encrypted at rest using AES-256
- TLS 1.3 encryption for data in transit
- API keys stored in secure vaults

### Access Controls
- Role-based access control (RBAC)
- Multi-factor authentication (MFA) for administrators
- Audit logging for all data access

### Data Retention
- Chat messages: Retained for 2 years
- Usage analytics: Retained for 1 year
- Deleted data permanently removed within 30 days
